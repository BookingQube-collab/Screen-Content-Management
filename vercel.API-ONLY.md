# Root `vercel.json` — API project only

This file configures **Project 1 (Express API)** when the Vercel **Root Directory** is the repository root (`.`).

It does **not** build or serve the Urban Arena React UI (`dist/public`). Opening the API deployment URL in a browser shows health JSON only.

For the kiosk and admin UI, create **Project 2** with Root Directory `artifacts/urban-arena`. See [DEPLOY.md](DEPLOY.md).
