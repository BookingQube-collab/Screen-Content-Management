Build a production-style web application for Urban Arena with two parts:
	1.	Public Display App
	2.	Admin Panel

The application is designed for a 21.5-inch vertical portrait screen kiosk display.
It must be lightweight, fast, modern, responsive, and easy to manage.

⸻

Core Goal

Create a media-based promotional display system where the admin can upload and manage:
	•	images
	•	videos
	•	activity name
	•	short description
	•	age limit
	•	terms & conditions

The public display should automatically show this content in a premium cinematic UI with:
	•	bold hero background media
	•	centered focus cards / slider
	•	Urban Arena branding
	•	age restriction badge
	•	terms and conditions text
	•	vertical portrait layout optimized for kiosk screen

⸻

Tech Stack Requirements

Use the latest lightweight stable technologies.

Frontend
	•	Next.js 15 with App Router
	•	React
	•	TypeScript
	•	Tailwind CSS
	•	Framer Motion for smooth animations
	•	ShadCN UI for admin components where useful

Backend
	•	Use Next.js server actions / route handlers
	•	Keep backend simple and lightweight
	•	Use Prisma ORM

Database
	•	PostgreSQL if available on Replit
	•	If setup becomes heavy, use SQLite first but keep schema Prisma-compatible for PostgreSQL migration

Storage
	•	Support upload for:
	•	image files
	•	video files
	•	Store files in a simple local uploads folder initially
	•	Architecture should allow future migration to cloud storage like S3 or Cloudinary

Authentication
	•	Simple admin login
	•	Secure session-based authentication
	•	Only admin users can access admin panel

⸻

Public Display App Requirements

Create a vertical portrait display page, such as:

/display

This page should look premium and modern, inspired by high-end promotional slider UI.

Public Screen UI Layout

Top Section
	•	Urban Arena logo
	•	optional large faded background word like “EXPLORE” or dynamic activity name
	•	glowing cinematic background effect

Hero Background Section
	•	full-width media background
	•	can be image or autoplay muted looping video
	•	should feel like background, not a small card
	•	overlay with gradient and darkening for readability

Foreground Focus Slider
	•	3-card or multi-card focus carousel
	•	center card active and large
	•	side cards partially visible with scale and perspective
	•	smooth transitions
	•	auto-slide every few seconds
	•	support both image and video thumbnails if possible

Each activity card can show:
	•	activity name
	•	short description
	•	CTA-style button like “Explore Now” or “Learn More”

Footer Section
	•	age badge like 18+, 16+, 7+, etc.
	•	terms & conditions text
	•	clean premium progress indicators / dots / line

Display Behavior
	•	autoplay slider
	•	full-screen friendly
	•	optimized for portrait screen ratio
	•	minimal CPU usage
	•	preload media smartly
	•	graceful fallback if video fails
	•	no unnecessary heavy effects that slow kiosk performance

⸻

Admin Panel Requirements

Create admin routes like:
	•	/admin/login
	•	/admin/dashboard
	•	/admin/activities
	•	/admin/activities/new
	•	/admin/activities/[id]/edit

Admin Features

Admin should be able to:

1. Manage Activities
Each activity must have:
	•	activity name
	•	slug
	•	short description
	•	full description
	•	age limit
	•	terms & conditions
	•	active/inactive status
	•	sort order
	•	featured flag

2. Upload Media
For each activity allow:
	•	hero background image
	•	hero background video
	•	slider card image
	•	optional thumbnail
	•	file preview before save

3. Control Display Content
Admin can:
	•	choose which activities appear on public display
	•	reorder activities
	•	mark one as default featured activity
	•	control autoplay sequence

4. Edit Branding
Admin should be able to update:
	•	Urban Arena logo
	•	brand colors
	•	footer text
	•	overlay heading like “EXPLORE”

5. Dashboard Summary
Show:
	•	total activities
	•	active activities
	•	images uploaded
	•	videos uploaded
	•	last updated items
